# Portfolio Landing Page Design Specification

## 1. Overview
A dark-themed, high-contrast data analytics portfolio page using neon green accents. The design combines dashboard UI patterns with a personal portfolio layout to communicate technical capability and impact quickly.

---

## 2. Design System

### Colours
- Primary Accent: #B6FF00 (neon green)
- Secondary Accent: #9CFF2E (glow variation)
- Background Base: #050A05
- Background Gradient: #050A05 → #0A1A1A
- Text Primary: #FFFFFF
- Text Secondary: #A0A0A0

### Effects
- Glow: soft neon green outer glow (blur 8px to 20px, opacity 40–70%)
- Background overlays: 5–15% opacity (code, UI fragments)

---

## 3. Typography

### Headings
- Style: Bold, condensed sans-serif
- Weight: 700–900
- Usage: Titles, section headers, name

### Body Text
- Style: Clean sans-serif (Inter / Poppins style)
- Weight: 400–500
- Line height: 1.4–1.6

### Numeric Emphasis
- Larger font scale for KPIs (2x body size minimum)

---

## 4. Layout System

### Grid
- 12-column grid
- Max width: 1200px–1400px
- Gutter: 16px–24px
- Section spacing: 80px–120px vertical

### Alignment
- Left-aligned content blocks
- Right-side supporting visuals

### Flow Pattern
Z-pattern scanning:
1. Name
2. Metrics
3. About
4. Skills
5. Experience
6. Work
7. CTA

---

## 8. Design Principles Applied

- High contrast for readability
- KPI-first information hierarchy
- Grouped skill categorisation
- Visual proof via dashboard preview
- Minimal but dense layout

---

## 9. Responsiveness (Guidelines)

### Tablet
- Reduce hero font size
- Stack sections vertically

### Mobile
- Single column layout
- Hide complex background visuals
- Reduce glow intensity

---

## 10. Suggested Enhancements

- Add micro-animations to pipeline graphic
- Introduce hover tooltips for skills
- Improve CTA with stronger button contrast
- Reduce background noise on smaller screens
