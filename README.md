# Data Analytics Portfolio

This repository contains the source code for my personal portfolio, built with Next.js and themed for a high-contrast data analytics aesthetic.

## Project Structure
- `/site`: The Next.js application source code.
- `Portfolio Landing Page Design Specification.md`: The design system and layout specs.
- `screen.png`: Screenshot of the design.

## Tech Stack
- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Charts**: Recharts
- **Icons**: Lucide React

## Deployment
This project is optimized for deployment on Vercel. 
**Note**: Set the root directory to `site` in your deployment settings.
