# Data Analytics Portfolio 📊

A premium, high-fidelity portfolio built with **Next.js**, **Tailwind CSS**, and **Framer Motion**. Designed to showcase data analytics expertise through a technical, data-driven aesthetic.

## ✨ Key Features

- **High-Frequency Hero**: Interactive terminal feed and a live "Engine Core" processing graphic.
- **Procedural Backgrounds**: Dynamic circuit board layouts generated on the fly.
- **Code Stream Overlay**: Real-time scrolling SQL, Python, and Metrics streams that react to interaction.
- **Data-Driven Stats**: Large metric displays highlighting key achievements and impact.
- **Premium Aesthetics**: Neon-accented dark mode with glassmorphism and scanline effects.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `src/components/Hero.tsx`: High-impact landing section.
- `src/components/Skills.tsx`: Procedural background with interactive code streams.
- `src/components/CodeStream.tsx`: Reusable scrolling text engine.
- `src/components/CircuitBackground.tsx`: Randomized SVG circuit generator.
